# Raman Project

Raman surface mapping produces large volumes of spectral data that are difficult to interpret manually. This work develops a framework to address this. A spatial visualiser is implemented to allow interactive exploration of Raman data. Principal component analysis and noise-whitened Harsanyi-Farrand-Chang method are investigated was estimating the number of distinct spectra present, and a non-negative factorisation technique is implemented to extract them. Lastly, a convolutional neural network proposed by Liu et al. as a unified Raman classifier is reconstructed, achieving a test accuracy of 90.7\%.
